== Description ==

Custom shortcodes and widgets built to enhance the [Clarity](http://www.madebyguerrilla.com/clarity-wordpress-theme) WordPress theme from [Blog Theme Machine](http://www.blogthememachine.com)

More details on specific widgets and shortcodes will be published here as soon as the write up is complete. For now, you can check out the current widgets and shortcodes offered below.

### Widgets
* Clarity About Box
* Clarity Popular Posts
* Clarity Social Icons

### Shortcodes
* `[btn]`
* `[recent-posts]`
* `[recent-work]`
* `[infobox]`
* `[related-posts]`

[view demo](http://www.madebyguerrilla.com/clarity-wordpress-theme)

== Installation ==

1. Upload the 'clarity-toolkit' folder to the '/wp-content/plugins/' directory or add it directly in your WordPress admin panel
2. Activate the plugin through the 'Plugins' menu in WordPress
